# ai-smart-traffic-light-system
An AI-based Smart Traffic Light System that dynamically adjusts traffic signal timing based on real-time vehicle density using Computer Vision. The system uses OpenCV to detect vehicles from video input and automatically increases or decreases signal timing depending on traffic congestion.

An AI-powered traffic management system that dynamically adjusts signal timing based on vehicle density using Computer Vision.

## 🔍 Features
- Real-time vehicle detection
- Traffic density analysis
- Dynamic green light timing
- Video/Webcam support

## 🛠 Tech Stack
- Python
- OpenCV
- NumPy

## 🚀 How to Run

1. Clone the repository
2. Install dependencies:

pip install -r requirements.txt

3. Run:

python traffic_light.py

Press 'q' to exit.
